Головна Users Posts Comments

Список завданнь

1. Зробити сторінку із довільним контентом і переходами на 3 сторінки:
  • users
  • posts
  • comments
2. На сторінці users через input[type=number] отримувати id юзерата та по кліку на кнопку Send вивести юзера в HTML через fetch та XMLHttpRequest. Стилізувати відповідним чином. https://jsonplaceholder.typicode.com/users/
3. На сторінці posts вивести в html пост через input[type=number] отримувати id поста та по кліку на кнопку Send вивести пост в HTML через fetch та XMLHttpRequest. Стилізувати відповідним чином. https://jsonplaceholder.typicode.com/posts
4. На сторінці comments вивести список коментарів через input[type=number] отримувати id коментаря та по кліку на кнопку Send вивести пост в HTML через fetch та XMLHttpRequest. Стилізувати відповідним чином. https://jsonplaceholder.typicode.com/comments
5. Візьміть будь яку лекцію по ООП, додайте застосування call, apply, bind
